# QuemAceitaBitcoin
Descubra e conheça as principais empresas do mundo que já aceitam Bitcoin!

###Acompanhe o andamento do projeto em https://github.com/btcbrdev/QuemAceitaBitcoin/wiki.###

## Install

```sh
$ [sudo] npm install
```

run server:

```sh
$ gulp
```

Deploy to gh-pages:

```sh
$ gulp deploy
```
